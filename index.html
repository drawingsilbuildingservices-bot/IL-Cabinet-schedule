Real-time sync across your whole team using Firebase (free). Set up takes about 5 minutes the first time.

1
Create a free Firebase project Go to console.firebase.google.com → Add project → name it → Create.
2
Enable Realtime Database Build → Realtime Database → Create Database → pick region → test mode → Enable.
3
Set rules to open Realtime Database → Rules tab → replace with {"rules":{".read":true,".write":true}} → Publish.
4
Get your Database URL & API Key Database URL is on the Data tab (e.g. https://project-rtdb.firebaseio.com). API Key is in Project Settings → Your apps → </> → Register.
Free Firebase Spark plan — 1 GB storage, 10 GB/month transfer. No credit card needed.

Each team member opens this page and enters the same URL and API key once.
