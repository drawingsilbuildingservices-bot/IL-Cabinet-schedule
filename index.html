Cabinet Schedule

Real-time sync across your whole team using Firebase (free). Set up takes about 5 minutes the first time — everyone on the team then just opens the same HTML file.

1
Create a free Firebase project Go to console.firebase.google.com → "Add project" → name it (e.g. cabinet-schedule) → click through → Create project.
2
Enable Realtime Database Left menu → Build → Realtime Database → Create Database → pick any region → start in test mode → Enable.
3
Copy your Database URL At the top of the Realtime Database page you'll see: https://your-project-default-rtdb.firebaseio.com — copy it below.
4
Copy your Web API Key Gear icon → Project settings → scroll to "Your apps" → click </> (web) → Register app → copy the apiKey value.
Free Firebase "Spark" plan: 1 GB storage, 10 GB/month transfer — more than enough for a cabinet shop. No credit card needed.

Save these credentials — share the same HTML file with your team. Each person enters the same URL and API key once.
