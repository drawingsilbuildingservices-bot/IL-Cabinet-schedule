2
Enable Realtime DatabaseBuild → Realtime Database → Create Database → pick region → test mode → Enable.
3
Set rules to openRealtime Database → Rules tab → replace with {"rules":{".read":true,".write":true}} → Publish.
4
Get URL & API KeyDatabase URL on the Data tab. API Key: Project Settings → Your apps → </> → Register.
